# v3.3.0 — Command Discovery Fix

**Date:** 2026-03-20

## What Changed

### Fixed
- **Commands not discovered by Claude Code** — Root cause: `plugin.json` explicitly declared `skills` and `agents` paths but omitted `commands`, preventing auto-discovery. Fix: removed all explicit component path declarations, letting Claude Code auto-discover `commands/`, `skills/`, and `agents/` from standard directory names.
- **Renamed `skill.md` to `SKILL.md`** in all 20 skill folders to match the official Claude Code plugin specification (case-sensitive filename requirement).
- **Extracted MCP server config** from inline `plugin.json` to standard `.mcp.json` file at plugin root.
- **Slimmed `marketplace.json`** to avoid conflicting metadata declarations with `plugin.json`.
- **Verified command frontmatter** — ensured all 6 command files have proper YAML frontmatter for discovery.
